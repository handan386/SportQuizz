It tests the user's sports knowledge through a series of multiple-choice questions with real-time feedback.

Key Features:
Multiple Choice Questions - Users choose from three different answers for each question.
Instant Feedback - Buttons change color to green for correct answers and red for wrong ones.
Progress Tracking - A progress bar shows how much of the quiz is completed, and the score is updated instantly.
Game Over Alerts: A pop-up message appears at the end of the quiz to show the final score and offer a restart.

Logic:
The app uses a QuizBrain structure to manage questions and logic. When a user taps an answer, the app checks if it is correct and updates the UI colors. After the final question, an alert displays the total points and allows the user to try again or exit.

Tech Details:
Language - Swift
Framework - UIKit
Architecture - MVC (Model-View-Controller)
